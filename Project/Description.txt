Данная БД создана на основании логики портала электронных объявлений Avito. 
Хранит в себе массив объявлений, диалогов и сообщений пользователей, а также необходимые технические таблицы, справочники, хранимые процедуры, обеспечивающие функионирование сервиса.
Используя встроенную процедуру возможен контроль истечения срока активности объявлений.

БД Включает в себя следующие таблицы:

ads - основная витрина объявлений
v_ads_detail - вью, содержащая более детальную информацию по объявлениям
categories - справочник категорий
cities - справочник городов
dialogs - витрина диалогов
favorites_ads - витрина избранныъ объявлений пользователей
feedbacks - витрина отзывов
messages - витрина сообщений
notify - параметрическая таблица настроек пользователей
phones - таблица телефонов
photos - таблица фотографий из объявлений
profiles - витрина атрибутов пользователей
users - основная витрина опльзователей
v_users_detail - вьюшка с детализацией по пользователям
views - таблица просмотров

Установочные файлы содержат в себе:
1. Description.txt - описание системы и решаемых ее задач
2. Create tables.sql - список ddl запросов для формирования необходимых таблиц. 
3. idx.sql - содержит связи таблиц по внешним ключам, а также необхоодимые индексы для частых запросов.
4. ERDiagram v2.png - диаграмма БД.
5.1. fulldb-clear.sql - файл с наполнением данным для БД
5.2. Operations.sql - ряд скриптов, корректирующих сгенерированный данные для предоставления им большей достоверности.
6. Queries.sql - типичные аналитические запросы к БД
7. Views.sql - Команды создание представлений для удобства просмотра детализированной информации
8. Functions.sql - Содержит в себе хранимые процедуры, функции и триггеры:
	а) t_b_del_phones - триггер, проверяющий и ограничивающий возможность удаления последнего номера телефона пользователя.
	б) f_last_phone - функция, выполняющая основную логическую часть для триггера а).
	в) p_mark_expired_ads - процедура, сканирующая активные объявления с последней датой изменения более 30 дней назад и маркирующая их статусом Expired
